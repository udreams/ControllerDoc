# 遥控器

遥控器采用左手油门（美国手）
![遥控器说明](https://raw.githubusercontent.com/udreams/ControllerDoc/master/Transmitter/Futaba.png)

注意：首先设置遥控器的模型为多旋翼模型，然后按如下设置遥控器的各个通道。由于EndPoint是相对于Sub-Trim的，因此油门的两个EndPoint相等才能对称。

对于四旋翼：

①AIL(Aileron)

②ELE(Elevator)

③THR(Throttle)

④RUD(Rudder)

⑤MODE

⑥TILT

⑦PAN

⑧REC(Recording)

⑨AUX(Auxiliary)
