# 快速入门

## 端口定义

| 名称      | 约定                                 |
|---------|------------------------------------|
| 基站和船控数据 | 14550                              |
| 伴随处理器接口 | 14540~14549，超过10个，后面的实例都用最后一个14549 |
| 流媒体     | 5600（H264.5）、5601（H265）            |

## 机器人ID规定

| 机器人ID号    | 约定              |
|-----------|-----------------|
| 0         | 内部预留，该ID号的性质为广播 |
| 1~100     | 给内部使用           |
| 101~65535 | 给客户使用           |
